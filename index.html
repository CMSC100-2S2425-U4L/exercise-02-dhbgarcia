RECENT PROJECTS

Silly Pedestrian Game

Silly Pedestrian is a JavaFX-based game where two Karts race infinitely across the screen, avoiding collisions with two randomly moving pedestrians. The game includes a 2-minute timer, collision detection, and a lives counter, with a limit of three collisions per Kart. Built with Java and JavaFX, the game features smooth animations, random pedestrian movement, and interactive UI elements for an engaging experience.

Kasangga

Kasangga is a web application designed to streamline partnership management and enhance communication between organizations and individuals in UPLB. It addresses the common challenges in partnerships between student organizations, such as delayed approvals, miscommunication, unaccountability, and unclear processes, ensuring smoother collaboration and more efficient workflows among all stakeholders.

Diet Problem Solver

The Diet Problem Solver is an R program that uses linear programming to find the most cost-effective food combination meeting daily nutritional needs. It minimizes cost while satisfying constraints on calories, vitamins, and minerals. Users can select food items and servings, and view the optimal solution with a table of food items, servings, cost, and a step-by-step tableau.

Tekknoblade

Tekknoblade 1v1 2D fighting game built with JavaFX, featuring real-time player movements, attacks, and blocking. It includes scenes for the menu, instructions, developers, game stage, and winner screen. Key features are character states, a timer, HP display, and OOP design with classes like GameTimer and Sprite. The game focuses on smooth performance, intuitive controls, and clear documentation.

P2P WannaBe

P2P WANNABE is a proposed solution to address the disorganization and security concerns in the ‘UPLB P2P’ Facebook group. It aims to create a dedicated website for UPLB students to easily access and review courses and professors. The platform will feature an organized data structure, allowing reviews to be categorized and indexed for efficient searching and filtering, while ensuring a secure, student-only environment.

Mario Typing Game

Mario-Themed Typing Game is a terminal-based Python game where players type random 3, 4, and 5-letter words for points. Scores are based on word length, and the leaderboard tracks the top 10 players. Players can view misspelled words after completing the game. The leaderboard is saved to a file and loaded each time the game starts.
